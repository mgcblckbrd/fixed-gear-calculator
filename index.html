Skid Patch Calculator

Калькулятор для fixed gear: посчитай количество скидпатчей (точек износа покрышки при торможении юзом), передаточное число, эквивалентные комбинации звёзд и скорость при заданном каденсе. Работает для любых покрышек, включая 700c.

О калькуляторе

Что это за инструмент

Skid Patch Calculator — это специализированный калькулятор для велосипедов с фиксированной передачей (fixed gear). Он помогает рассчитать ключевые параметры трансмиссии: количество скидпатчей, передаточное число, скорость при заданном каденсе и эквивалентные комбинации звёзд.

Что такое скидпатчи (skid patches)

На фиксе торможение часто происходит юзом — заднее колесо блокируется и скользит по асфальту. При каждом юзе стирается одна и та же точка на покрышке. Количество скидпатчей — это количество разных точек на покрышке, которыми вы будете тормозить. Чем их больше, тем равномернее изнашивается резина и тем дольше она живёт.

Количество скидпатчей зависит от соотношения зубьев передней и задней звезды. Калькулятор ищет наибольший общий делитель (НОД) и вычисляет количество уникальных позиций колеса при торможении.

Что такое Ambidextrous skid

Обычный юз выполняется одной ведущей ногой. Ambidextrous skid — это техника, при которой велосипедист умеет тормозить юзом с обеих ног, меняя ведущую. Это удваивает количество скидпатчей и значительно продлевает жизнь покрышке. Калькулятор учитывает это при расчёте.

Что считает калькулятор

  • Skid Patches — количество точек износа при обычном и амбидекстральном торможении.
  • Ratio — передаточное число (отношение передней звезды к задней).
  • Equivalent Gear ±2% — альтернативные комбинации звёзд, которые дают почти такое же передаточное число, но с другим количеством скидпатчей. Полезно для подбора оптимальной трансмиссии.
  • Cadence / Speed — таблица соответствия каденса (оборотов педалей в минуту) и скорости. Можно узнать скорость при привычном каденсе или понять, какой каденс нужен для желаемой скорости.

Как пользоваться

  1. Введите количество зубьев передней звезды (chainring).
  2. Введите количество зубьев задней звезды (cog).
  3. Выберите размер покрышки из списка.
  4. Отметьте галочку Ambidextrous skid, если владеете этой техникой.
  5. Нажмите «Рассчитать».

Результаты появятся мгновенно. Никакие данные не отправляются на сервер — всё считается прямо в браузере.
