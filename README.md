# Skid Patch Calculator

Калькулятор для велосипедов с фиксированной передачей (fixed gear). Рассчитывает количество скидпатчей, передаточное число, скорость при заданном каденсе и эквивалентные комбинации звёзд.

## 🚀 Возможности

- Расчёт скидпатчей (skid patches) с учётом ambidextrous skid (амбидекстральное торможение)
- Передаточное число (ratio)
- Таблица Cadence / Speed
- Поиск эквивалентных комбинаций звёзд (±2%)
- Поддержка разных размеров покрышек

## 🛠 Технологии

- HTML
- CSS
- Чистый JavaScript

## 📦 Как запустить

1. Скачайте или клонируйте репозиторий.
2. Откройте `index.html` в браузере.
