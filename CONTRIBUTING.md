# Правила участия

Спасибо, что хотите помочь проекту! Вот несколько простых правил.

## Как предложить улучшение

1. Создайте [issue](https://gitverse.ru/mgcblckbrd/fixed-gear-calculator/issues) с описанием бага или предложения.
2. Дождитесь обсуждения — возможно, мы решим, что фича не подходит проекту.
3. Если идея одобрена — делайте fork и создавайте pull request.

## Как оформить pull request

- Один PR = одна фича или один багфикс.
- Проверьте, что код работает в последних версиях Chrome и Firefox.
- Следуйте стилю кода, который уже есть в проекте.

## Стиль кода

- HTML: семантические теги, отступы 2 пробела.
- CSS: логические свойства, классы в kebab-case.
- JS: ES6+, `const` по умолчанию, функции объявлять через `function`.
